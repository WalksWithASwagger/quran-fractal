# Quran Fractal Web App

Interactive Next.js explorer for the verification artifacts generated by the Python pipeline.

## Prerequisites

- Node.js 20+
- JSON data generated at `public/data/quran-data.json`

Generate fresh data from the repository root:

```bash
python3 verify.py
```

## Local Development

```bash
npm ci
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks

```bash
npm run lint
npm run build
npm audit --omit=dev
```

## Data Contract

The app reads `public/data/quran-data.json` and does not re-implement counting logic in TypeScript.
See `../docs/api-contract.md` for field-level details.
